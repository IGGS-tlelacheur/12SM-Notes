Revision of Proof

Quantification

For all (∀)

A universal statement claims that a property holds for all members of a given set. Such a statement can be written using the quantifier ‘for all’. For example:

For all real numbers x and y, x2+5y2⩾2xy or ∀x,y∈R (x2+5y2⩾2xy)

To prove this a general argument that applies for all choices of x and y is required.

There exists (∃)

An existence statement claims that a property holds for at least one member of a given set. Such a statement can be written using the quantifier ‘there exists’. For example:

There exists a triple of integers a,b,c such that a2+b2=c2 or ∃a,b,c∈Z (a2+b2=c2)

To prove that this statement is true, we just need to give one instance.

Negation of Quantifiers

When negating a statement with a quantifier, the quantifiers must also change.

  • ‘There exists’ becomes ‘For all’
  • ‘For all’ becomes ‘There exists’

Example 1.1 Write down the negation of each of the following statements:

(a) For all natural numbers n, we have 2n⩾n+1

(b) There exists an integer m such that m2=4 and m3=8

(c) For all real numbers x and y, if x<y, then x2<y2

hint: ¬(P⟹Q)=P∧¬Q

Disproving Statements

To disprove:

  • a universal statement, a single counterexample is required
  • an existence statement, a proof of the negation is required (which is a universal statement)

Example 1.2 Disprove the following:

(a) For all real numbers n, n2 is either a multiple of 2 or 3.

(b) There exists a natural number n such that n2+3n+2 is odd.

Contradiction

The basic outline of a proof by contradiction is:

  1. Assume that the statement we want to prove is false.
  2. Show that this assumption leads to mathematical nonsense.
  3. Conclude that we were wrong to assume that the statement is false.
  4. Conclude that the statement must be true

Example 1.6 Suppose x satisfies 2x=5, prove, by contradiction, that x is irrational.

Equivalent Statements

P⟺Q

It can also be said that P is true if and only if Q is true.

This statement states P⟹Q and Q⟹P. To prove an equivalence, both statements must be proven.

Example 1.7 Let n∈Z, prove that n is divisible by 3 if and only if n2 is divisible by 3.
